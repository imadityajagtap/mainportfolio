# Aditya Jagtap — Personal Portfolio

Personal portfolio website showcasing my MBA journey, case studies, research, 
and insights in finance, strategy, and consulting.

## Features
- Public portfolio site with projects, research, blog, contact
- Private admin panel with full CRUD capabilities
- Dark/Light mode toggle
- Fully responsive with smooth animations
- Custom cursor and micro-interactions

## Tech Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS + Shadcn/ui
- MongoDB + Mongoose
- NextAuth.js
- Framer Motion + Lenis
- Cloudinary

## Getting Started
1. Clone repo
2. Run `npm install`
3. Copy `.env.example` to `.env.local` and fill credentials
4. Run `npm run dev`
5. Visit http://localhost:3000

## Admin Panel
Access at `/admin/login`
